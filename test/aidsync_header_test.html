AidSync Header Demo

This is a fully functional demo of your AidSync header component. The header includes:

  • Responsive design - Adapts to different screen sizes
  • Dark mode toggle - Click the moon icon to switch themes
  • Mobile menu - Hamburger menu appears on smaller screens
  • Smooth transitions - All interactions are animated
  • Accessibility features - Proper ARIA labels and keyboard navigation
  • Sticky header - Stays at the top when scrolling

Testing Instructions

Desktop: Try hovering over navigation links and buttons to see the hover effects.

Mobile: Resize your browser window or use developer tools to test the mobile menu.

Dark Mode: Click the moon icon in the top right to toggle between light and dark themes.

Accessibility: Try navigating with the Tab key to test keyboard accessibility.

Features Working

✅ Dark mode toggle with localStorage persistence

✅ Mobile hamburger menu with smooth slide animation

✅ Responsive navigation that hides/shows based on screen size

✅ Click outside to close mobile menu

✅ Accessibility features (ARIA labels, keyboard navigation)

✅ Smooth transitions and hover effects
